**11/9**

우리 포춘디어 멤버들은 항상 목요일에 회의를 한다.

내가 회의 목차와 일정을 알려주면 다들 준비해온다.

그리고 이번 목차는


```
1.기능명세서&API명세서 수정

2.대표캐릭터 아이디어 수정

3.프론트와 백엔드 문제 논의
```

였다

1번과 3번은 잘 되었다.

근데 항상 문제는 대표 캐릭터이다.

대중성 따지고 세세한 컨셉을 추가 수정을 반복한다.

그래서 이번에 또 바뀌었다

```
포춘쿠키 모양 머리핀 -> 포춘쿠키 모양 귀걸이
```

항상 포춘쿠키를 캐릭터와 연관시키는 것이 힘들었다.

모자, 핸드백, 귀걸이, 머리핀, 옷 등

다양한 선택지 중 다른 사람들(친구들)을 투표에 참가시켜

결국 귀걸이로 바꾸었다.

-정보-

요즘 학교에서 내는 과제가 너무 많다 - ex) java, 자료구조 프로젝트, ppt 정리 및 발표 등

요즘 연말이라 그런지 모두 바뻐 포춘디어의 진행 상황이 점점 느려져 가고 있다.

그리고 우리는 내년 즉 2024년 6월달 까지 포춘디어를 완성시키기로 하였다.

그리고 나머지 6개월은 유지 및 보수를 할 것이고

지금은 html을 배우지만 1월달 부터 React를 사용해 만들 예정이다.

회의(나의 통보?)를 통해 결정된 사안이다.
